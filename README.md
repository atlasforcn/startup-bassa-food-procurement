# BASSA一站式商業食材採購平台 Demo

這個 repo 是依據教育部「大專校院創新創業實戰模擬學習平臺」成功案例建立的原生前端互動 demo，主題為國立中正大學團隊 002FiatLux 的 `BASSA一站式商業食材採購平台`。

## 核心概念

BASSA 是面向商業餐飲業者的食材採購平台，核心價值是把供應商、食材品項、即時報價、採購單、配送安排與庫存/成本資訊整合在同一個工作流程中，協助餐廳降低採購溝通成本並掌握營業成本。

## Demo 範圍

本 demo 以餐廳採購營運台為範圍，包含：

- 食材目錄與分類搜尋
- 供應商報價比較與最佳報價提示
- 採購車與採購單金額計算
- 配送排程與未排程提醒
- 餐廳庫存水位、低庫存警示與成本分析
